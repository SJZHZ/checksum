7. 内部Trivial外部Alder
不可能保证同态。
证明：
    设 C(1 << i) = Table[i]
    由同态性，C(1 << (i + 1)) = Table[i + 1] = 2 * Table[i] = Table[i] << 1。
    即Table[k] = Table[0] << k, 高位信息丢失。
